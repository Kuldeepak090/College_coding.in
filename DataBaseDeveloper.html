Course Curriculum

Introduction to Databases:
  • What is a database and why do we need it?
  • Types of databases: relational, NoSQL, graph, etc.
  • Components of a database system: schema, data, query language, etc.
  • Data modeling: entity-relationship diagrams, UML, etc.
Relational Data Model
  • Introduction to the relational model
  • Tables and relations
  • Primary and foreign keys
  • Normalization: first, second, and third normal form
  • Functional dependencies and multi-valued dependencies
SQL (Structured Query Language)
  • Introduction to SQL
  • Data definition language (DDL): CREATE, ALTER, and DROP statements
  • Data manipulation language (DML): SELECT, INSERT, UPDATE, and DELETE statements
  • Query optimization and performance tuning
Advanced SQL
  • Aggregate functions and grouping
  • Subqueries and joins
  • Views and materialized views
NoSQL Databases
  • Introduction to NoSQL databases
  • Types of NoSQL databases: key-value, document, column-family, graph, etc.
  • Comparison with relational databases
  • Use cases and examples
Big Data and Distributed Databases
  • Introduction to big data
  • Distributed database systems
  • MapReduce and Hadoop
  • Distributed query processing
Data Warehousing and OLAP
  • Introduction to data warehousing
  • Star and snowflake schemas
  • OLAP cubes and dimensions
  • Data mining and machine learning
Data Integrity and Security
  • Data integrity constraints
  • Database security
  • Authentication and authorization
  • Auditing and logging
